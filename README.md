# 🧬 SOVEREIGN EMPIRE - 12 AGENT DNA ECOSYSTEM

Complete autonomous AI content generation system with 12 specialized agents inheriting from a single DNA blueprint.

## 📊 AGENTS

1. **Content Generator** - Blog posts, articles
2. **SEO Optimizer** - Search engine optimization
3. **Social Media Distributor** - Multi-platform posting
4. **Email Writer** - Campaign emails
5. **Headline Generator** - Viral titles
6. **Keyword Researcher** - SEO keywords
7. **Competitor Analyzer** - Competitive intelligence
8. **Trend Analyzer** - Emerging trends
9. **Podcast Script Writer** - Audio content
10. **Video Script Writer** - Video content
11. **Revenue Tracker** - Financial metrics
12. **Performance Optimizer** - Auto-optimization

## 🚀 QUICK START

```bash
# Install dependencies
npm install

# Add your API keys to .env
# ANTHROPIC_API_KEY=sk-ant-...
# SUPABASE_URL=https://...
# SUPABASE_KEY=...

# Test all agents
node test-all-agents-dna.js

# Deploy
npm run start
```

## 💡 HOW AGENT DNA WORKS

All 12 agents inherit from `AgentDNA` class, which provides:
- **Execution** - Run tasks with retry logic
- **Learning** - Improve from past results
- **Self-healing** - Fix errors automatically
- **Metrics** - Track performance
- **Cost optimization** - Model routing
